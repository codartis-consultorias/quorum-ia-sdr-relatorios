# Relatorios de teste — Sistema Graces SDR

Publicacao automatica do harness de testes (`tests/run.js` do repo
`quorum-ia-sdr`). Cada rodada faz commit do relatorio HTML aqui.

- **index.html** — sempre o relatorio mais recente.
- **relatorio-<data>.html** — historico, um por rodada.

Visivel via GitHub Pages. Contem transcricoes dos contatos de teste.
